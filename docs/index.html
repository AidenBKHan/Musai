Musai

전 세계 어디든 확인할 수 있는 글로벌 안전지수 — 앱, 홈스크린 위젯, 그리고 다른 웹사이트에 바로 붙일 수 있는 웹 위젯으로 제공됩니다.

웹 위젯 미리보기

아래는 musai-widget.js를 그대로 임베드한 실제 결과입니다. 아직 백엔드가 공개 배포되지 않아 데모 모드(placeholder 데이터)로 표시됩니다.

다른 웹사이트에 적용하는 방법

<div class="musai-safety-widget"
     data-country="KR"
     data-region="Seoul"
     data-api-base="https://your-musai-backend.example.com"></div>
<script src="https://your-cdn.example.com/musai-widget.js" async></script>

자세한 옵션은 저장소의 web-widget/README.md를 참고하세요.

프로젝트 구성

  • app/ — Flutter 앱 + Android/iOS 홈스크린 위젯
  • backend/ — 국가별 안전 데이터를 표준 스키마로 합쳐주는 API
  • web-widget/ — 이 페이지에 쓰인 임베드 가능한 웹 위젯
