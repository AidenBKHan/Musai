바우처 확인 화면에 뜨는 안전정보 미리보기

목적지를 고르면, 여행자가 예약 완료 후 보게 될 것과 같은 위젯이 그대로 나타납니다. 백엔드가 아직 공개 배포되지 않아 지금은 데모 데이터(제안서의 파리 72점 예시 포함)로 표시됩니다.

위젯 형태 한눈에 보기

삽입 위치에 따라 다섯 가지 레이아웃을 고를 수 있어요 — 일반 카드, 모바일 예약완료 화면용 바텀시트, 데스크톱 사이드바용으로 길게 늘어진 와이드(뚱뚱한 박스가 아니라 한 줄로 길쭉하게 펼쳐지는 형태), 화면 구석에 떠 있다가 클릭하면 펼쳐지는 버블, 그리고 예약완료 화면 안에 놓는 CTA 배너(제안서 목업과 동일한 형태).

card — 일반 페이지 삽입용

bottomsheet — 모바일 예약완료 화면용

wide — 길게 늘어진 데스크톱 사이드바용

bubble — 화면 구석의 작은 아이콘, 클릭하면 펼쳐짐

banner — 예약완료 화면 안의 CTA 배너, 클릭하면 펼쳐짐

프로젝트 구성

  • app/ Flutter 앱 + Android/iOS 홈스크린 위젯
  • backend/ 국가별 안전 데이터를 표준 스키마로 합치는 API
  • web-widget/ 이 페이지에 쓰인 임베드형 웹 위젯 소스
