위젯 적용 가이드

여러분의 사이트(예: 여행 예약 사이트)에 Musai 안전지수 카드를 붙이는 방법입니다.

← 홈으로

1. 값 입력하고 스니펫 생성

나라 코드는 필수, 지역명은 화면에만 표시되는 선택 값입니다. API 서버 주소를 비워두면 배포 전에도 안전하게 데모 모드로 표시됩니다. 마스코트 이미지는 별도 파일이 아니라 musai-widget.js 안에 내장되어 있어서, 아래 스니펫의 script src만 실제로 접근 가능한 주소로 두면 이미지까지 그대로 따라옵니다.

2. 옵션 설명

속성필수설명
data-country필수목적지 ISO 3166-1 alpha-2 국가 코드 (예: FR, JP, KH)
data-region선택화면에 표시할 지역/도시명 (표시 전용, 비우면 백엔드 값 사용)
data-layout선택card(기본) · bottomsheet(바우처·예약완료 화면용) · wide(데스크톱 사이드바용, 큰 마스코트 이미지 포함) · bubble(평소엔 화면 구석의 작은 아이콘, 클릭하면 바텀시트로 펼쳐짐 — 부가 서비스처럼 방해되지 않게 노출하고 싶을 때)
data-api-base선택배포된 Musai 백엔드 주소. 비워두면 데모 모드로 표시

요청이 3초 안에 응답하지 않거나 실패해도 자동으로 데모 모드로 전환되어, 백엔드 상태와 무관하게 여러분의 페이지는 깨지지 않습니다.
